🔐 暗号化パスワードを入力

クラウドデータが暗号化されています。パスワードを入力して復号してください。

パスワードを忘れた場合

⚖ 藤野晃司完全管理システム

クラウド同期のため GitHub Token の設定が必要です（無料）。

手順：
① github.com/settings/tokens/new にアクセス
② Note に名前を入力し、「gist」にチェックを入れる
③「Generate token」で表示されたトークン（ghp_...）を下に貼り付け

設定するとクラウドに保存されるデータがAES-256で暗号化されます。忘れると復号できないため注意。

※ 設定は後から変更できます。「このPCのみ」を選んだ場合、データはこのブラウザのみに保存されます。
